## 使用说明

- dv  ----------------------------- (说明)
- dv -v --------------------------- (版本号)
- dv list ------------------------- (当前已存储模板列表)
- dv add  ------------------------- (新增模板列表模板列表)
- dv delete ----------------------- (删除当前已存储模板列表. 传入*将会初始化所有模板)
- dv init ------------------------- (根据已有模板创建项目)
- dv create ----------------------- (创建新的模板, 并且同时创建项目)
